### とりあえず実行方法

projectフォルダをダウンロード，projectディレクトリ下で以下を実行

コンパイル  javac -cp ".:gson-2.10.1.jar" *.java

サーバー起動  java -cp ".:gson-2.10.1.jar" Server

クライアント起動（別ターミナル）  java -cp ".:gson-2.10.1.jar" Client

### 動作確認の使い方の注意点
グループIDのところにはグループ名ではなく生成されたIDを入れる
